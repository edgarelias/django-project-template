Tips:
  1. If you don't have experience with Django, the best way to start is by reading the official documentation.
  2. Understand Python 3 or you will struggle.
  3. Django uses a lot of classes, you must have experience with OOP
  4. If you want to use Django as your backend and frontend, you must understand web forms
  5. If you want to use Django as your backend and React, Angular, or Vue as your front-end, you need to build your django app as a RESTful service
  6. Understand how to build a relational database. Django and its Object Relational Mapper relies on using a Relational Database
  7. If you do try to use Django, without understanding how it works you will waste hours
